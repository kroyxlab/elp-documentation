# Punctuation Marks

The Encapsulated Language uses the following symbols:

| Symbol  | Rules                                     | Example   |
| :-----: | ----------------------------------------- | --------- |
|         | A separator symbol for digit groups.      | 1 000 000 |
| .       | A separator symbol for dozenal fractions. | 1.0       |
